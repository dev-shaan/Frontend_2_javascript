## 📂 Featured Projects Blueprint

### 1. 🎲 Dicee Challenge
A classic two-player dice rolling game used to solidify foundational JavaScript control , random number generation, and fundamental DOM adjustments.
* **Core Learnings:** Dynamic image path strings, `Math.random()`, and conditional rendering logic based on score tracking.
* **Tech Stack:** HTML5, CSS3, JQuerry.

### 2. 🥁 Drum Kit
An interactive, virtual drum set application that triggers specific high-quality audio tracks through both on-screen clicks and physical keyboard presses. 
* **Core Learnings:** Keyboard event listeners (`keydown`), HTMLAudioElement manipulation, tracking `this` identity in event handlers, and adding temporary CSS animation states via JavaScript classes.
* **Tech Stack:** HTML5, CSS3, JavaScript (JQuerry).

### 3. 🎮 Simon Game
An interactive electronic memory sequence game where the application generates a shifting pattern of light and tone that the player must memorize and replicate. The difficulty scales dynamically with every successful round.
* **Core Learnings:** State management arrays (game pattern vs. user pattern), tracking complex game states (started vs. over), dynamic callback functions, and logic-driven array comparison loops.
*  **How to Play:** The game will light up and play a sound for one color at the start of each round. Your goal is to replicate the sequence from the very beginning. Every time you successfully repeat the pattern, the game adds one new random color to the end of the chain.You need to build the chain.
* **Tech Stack:** HTML5, CSS3, JavaScript, jQuery.
